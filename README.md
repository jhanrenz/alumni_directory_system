# Campus Alumni Directory System

## Description

A full-stack web application designed to streamline alumni management for educational institutions. The system enables administrators to manage user registrations, events, courses, and alumni records, while providing users access to an organized alumni directory and event listings.

It features an approval-based access system, ensuring controlled onboarding of users. The platform is actively maintained, with continuous feature enhancements and ongoing UI improvements.

## Tech Stack

**Backend:**

* Laravel (REST API)
* Laravel Sanctum (Authentication)
* DomPDF (PDF export functionality)

**Frontend:**

* React (Functional Components & Hooks)
* TypeScript
* TanStack Query (Server State Management)
* Axios
* Tailwind CSS

## Core Features

* User registration with admin approval workflow
* Role-based access control (Admin/User)
* Alumni directory browsing
* Event management system
* Course management
* Export alumni records to PDF
* Authentication & authorization

## Status

* Actively maintained
